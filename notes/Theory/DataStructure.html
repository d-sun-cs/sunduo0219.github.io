                                                            # Pattern matching:

                                                            • Violent matching: O(nm)O(nm)O(nm)
                                                            • KMP:
                                                              • Thoughts: Longest equal prefix and suffix / Prefix List / Partial Match List
                                                              • Steps:
                                                                1. Get next / lengths of longest equal prefix and suffix: Mathematical induction
                                                                  1. next[0] = 0
                                                                  2. Given next[j] = k, judge whether s[j + 1] == s[k]
                                                                  3. If s[j + 1] != s[k], k = next[k - 1], judge whether s[j + 1] == s[k], until k - 1 < 0
                                                                  4. If s[j + 1] == s[k], next[j + 1] = k + 1, continue
                                                                  5. next[j] = next[next[j]] until s[j] != s[next[j]]
                                                                2. Match from front to back with next[j - 1]
                                                              • Time complexity: O(n+m)O(n+m)O(n+m)
