                                                # 0.0 Concept

                                                • Concept:
                                                  • Describe the solving steps of specific problems
                                                  • Limited sequence of instructions
                                                • Features:
                                                  • Finiteness: finite steps, finite time of each step
                                                  • Certainty: certain meaning of each step, the same input, the same output
                                                  • Feasibility: each instruction can be implemented by basic operations
                                                  • Input: zero or multiple inputs
                                                  • Output: one or multiple outputs
                                                • Goals:
                                                  • Validity
                                                  • Robustness
                                                  • Readability
                                                  • Less Complexity

                                                # 0.3 Space Complexity

                                                • Concept:

                                                  • Describe the memory usage when program running by S(n)=O(g(n))S(n)=O(g(n))S(n)=O(g(n)), g(n)g(n)g(n) is the function of problem scale nnn

                                                    Not the size of executable file

                                                • Calculate:

                                                  • Relation between memory usage and problem scale nnn

                                                • Recursion:

                                                  • recursion levels

                                                    level, not times, but the maximum quantity of stack frames at a time

                                                  • space complexity each recursion

                                                • Memory consumption:

                                                  • Different languages:

                                                    • C/C++: require / release memory heap
                                                    • Java: JVM
                                                    • Python: private heap managed by interpreter

                                                  • Memory management of C/C++

                                                    • Fixed part:

                                                      • Text: binary code of functions
                                                      • Data: initialized or uninitialized global and static variables

                                                    • Variable part:

                                                      • Stack: auto allocated and released by complier

                                                      • Heap: allocated and released by programmer

                                                        If not, OS manage it at the end of the program

                                                  • Memory size: bytes of a pointer

                                                    • In 32-bit OS: 4 bytes, 4G memory
                                                    • In 64-bit OS: 8 bytes, more than 4G memory

                                                  • Memory alignment

                                                    • Why:

                                                      • platform: cross-platform, only data in specific address can be accessed in some platforms

                                                      • hardware: increase the speed of memory access

                                                        if not aligned, times of memory access will increase for concatenating not aligned data

                                                    • How: complier automatically do it
